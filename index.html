Sumatoria de fuerzas: Componentes

Ciencias Básicas ⭐ 3.9 (50 Reviews)
Lección 30 / 60
⏱️ 1 hora 24 minutos 📚 2 temas 📜 Docs

Sobre la lección

En esta lección, aprenderás a descomponer fuerzas en sus componentes rectangulares y cómo aplicar esta técnica para realizar sumatorias de fuerzas en diferentes direcciones (ejes X y Y). Este análisis es esencial para resolver problemas de equilibrio en estructuras y sistemas mecánicos.

Exploraremos:

  • Qué son los componentes de una fuerza.
  • Cómo usar trigonometría para obtener las componentes horizontal y vertical.
  • La importancia del signo (+/–) según el sistema de referencia.
  • Aplicación práctica en ejercicios de cuerpos en equilibrio.

Al finalizar, podrás representar correctamente cualquier fuerza como la suma de sus componentes y aplicar este conocimiento para resolver situaciones reales en estática y física general.
